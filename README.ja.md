# Barcoder — ストライプベース

**Languages:** [简体中文](README.md) · [English](README.en.md) · [日本語](README.ja.md)

[ELECTRONICOS FANTASTICOS!](https://www.electronicosfantasticos.com/en/) 風の**バーコードベース** Web シミュレーション。Processing **Barcoder** スキャンモデル：レーザーが白黒ストライプを走査 → 光電サンプリング → **ミラー往復（音声正反）** → リアルタイム発音。

## 2 つの演奏モード

### バーコード演奏

キャンバス上で **押したまま** ストライプに沿ってスキャン；**ホイール**で **スキャン線幅** を調整。

| バーコード形式 | 説明 |
|----------------|------|
| 8本の縦ストライプ | バーごとの密度；ベース練習 |
| ボックスグリッサンド | 横方向の周期変化；上下グリッサンド |
| ボックス放射状 | 中心から放射；2 次元の音高 |
| 画像をアップロード | PNG/JPG など；長条はホイールでスクロール |
| 音声→バーコード | 波形を横条に展開（**テスト**；下記） |

**音声→バーコード（テスト）：** WAV/MP3（≤10 秒）を選択し、**サンプル密度**（列/秒）と **列幅**（px/列）を設定 → **バーコード生成** / **PNG 出力**。プレビューは可能ですが、**ブラウザ内ではスキャンできません**（テストバナー参照）。列幅を広げると印刷後のスキャンが容易。

### 楽譜エディタ

**行ごとに作曲**；1 行 = 1 バーコード。**フレームレート（ミラー）** と **スキャン線幅** はロック（`f ≈ レート×線幅/周期`）；**スキャン方向** はいつでも切替可能。

- **縦ストライプ / グリッサンドストライプ** — 上段は縦と同密度；下へ滑らかに扇形
- **和音（テスト）** — 横条の **織り**；**織り回数**（例：2 音 ×2 → ABAB）
- **音名一括入力** — `G4 G4 A4 …` を貼り付け → **一括生成**；プレビューは楽譜と同期；**X** 固定行は赤表示
- **選択行を編集** — 行に合わせ **X** で編集固定；音高変更、**縦ストライプ** ↔ **グリッサンドストライプ**
- **MIDI 入力** — **MIDI 接続**；全キー離すと 1 行追加（同時押しは和音）
- **PNG 出力** — **音名付き** / **バーコードのみ**；**長条1枚** / **A4 分割印刷**
- **デモ：ハッピーバースデー**

**ホイール**で楽譜スクロール；**Delete** で行削除；ホバーで **リアルタイム音高**（クリック不要）。

## ミラー / スキャン

| 項目 | 既定 | 説明 |
|------|------|------|
| フレームレート（ミラー） | 22 Hz | **0** → ドラッグで手動スキャン |
| スキャン線幅 | 200 px | **0** → 点レーザー（移動軌跡をサンプル） |
| レーザー幅（移動平均） | 2 | スキャン線上の移動平均 |
| スムーズ Lerp | 0.5 | スキャン方向の平滑化 |
| ミラー往復（音声正反） | オン | Barcoder 式の往復波形 |
| Barcoder 窓関数 | オン | 区間の端をフェード |

**音高（フッター式）：** `f ≈ フレームレート × 線幅 / 周期`（Processing Barcoder）

楽譜の **縦ストライプ** は小数周期で目標音高に合わせます。**バーコード演奏** 側は整数ピクセルでくっきり描画します。

## 操作早見表

- **スキャン発音：** バーコード上で押したまま移動（線幅 > 0 で赤いスキャン線）
- **線幅：** ホイール（**楽譜エディタ** ではホイール = 楽譜スクロール）
- **スキャン方向：** マウス追従（移動に垂直） / 横方向ロック / 縦方向ロック
- **行固定：** **X**（もう一度で解除）
- **行削除：** **Delete**
- **長い画像：** ホイールまたは ↑↓
