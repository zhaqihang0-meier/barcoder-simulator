# barcoder-simulator
一个网页式barcoder模拟器，它支持对扫码枪的参数进行修改来调音。
网页有两个模块：
一、扫描模式，支持扫描预设的三种条码和上传的图片，也支持通过音频生成相应条码。
二、乐谱模式，该模式下锁定扫码枪参数，通过输入音高来生成相对应的条码乐谱，也支持接入midi设备输入。生成的乐谱可导出PNG文件

A web-based barcoder simulator that allows modification of the parameters of the scanner to adjust the tone.
The web page has two modules: 
1. Barcode play , which supports scanning of the preset three types of barcodes and uploaded images, as well as generating corresponding barcodes through audio. 
2. Score editor , in this mode, the scanner gun parameters are locked, and the corresponding barcode sheet music is generated by inputting the pitch. It also supports connecting to MIDI devices for input. The generated sheet music can be exported as a PNG file.

ウェブページ形式のババーコードシミュレーターで、スキャンペンのパラメータを調整して音色を調節できます。  
このウェブページには2つのモードがあります。  
1. バーコード演奏：プリセットされた3種類のババーコードやアップロードした画像をスキャン可能で、音声からから対対応するババーコードを生成することも可能です。  
2. 楽譜エディタ：このモードではスキャンペンのパラメータを固定し、音高を入力することですることで対対応するババーコード楽楽譜を生成します。また、MIDIデバイスからの入力をサポートしています。生成された楽楽譜はPNGファイルとしてエクスポート可能です。
