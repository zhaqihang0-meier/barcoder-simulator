# Barcoder — 条纹贝斯

**Languages:** [简体中文](README.md) · [English](README.en.md) · [日本語](README.ja.md)

[ELECTRONICOS FANTASTICOS!](https://www.electronicosfantasticos.com/en/) 风格的光电条码乐器 Web 模拟，基于 Processing **Barcoder** 扫描模型：激光扫过黑白条纹 → 光电采样 → 振镜正反拼接 → 实时发声。

**作者：** by [極東踊羊Meier](https://search.bilibili.com/all?keyword=%E6%A5%B5%E6%9D%B1%E8%B8%8A%E7%BE%8AMeier)（bilibili）

界面支持 **简体中文 / 繁體中文 / English / Español / 日本語**（页头语言切换）。

## 两种演奏模式

### 条码演奏

在画布上 **按住鼠标** 沿条纹方向扫描发声；**滚轮** 调节扫描线宽。

| 条码类型 | 说明 |
|----------|------|
| 八条竖纹 | 每条独立疏密，经典贝斯练习 |
| 方框滑音条码 | 横向周期渐变，上下滑音 |
| 方框放射纹 | 从中心向外放射，二维音高 |
| 上传图片 | PNG/JPG 等，长图可滚轮抽拉 |
| 语音转条码 | 音频波形 → 横向竖条（**测试功能**，见下） |

**语音转条码（测试）：** 可选 WAV/MP3（≤10 秒），调节 **采样密度（列/秒）** 与 **列宽（px/列）** 后生成条码并导出 PNG。可在网页预览，但 **暂不支持在浏览器内扫描播放**；列宽越大越适合打印后匀速扫过。

### 乐谱编写

逐行作曲，每行一条码；进入模式后 **振镜频率与扫描线宽锁定**（保证音高一致），振镜方向仍可切换。

- **竖纹 / 滑音条纹**：滑音顶段与竖纹同疏密，向下流线扩散或收拢
- **合音**：多声部交错条纹（可设交错次数）
- **批量输入音名**：粘贴 `G4 G4 A4 …` 一键生成；上方实时预览条随乐谱同步，**X 锁定行** 时对应音名变红
- **编辑选中行**：鼠标指向某行，按 **X** 锁定；可改音高、竖纹 ↔ 滑音
- **Web MIDI**：连接键盘录入（和弦一次抬起写入一行）
- **导出 PNG**：带/不带音名；**长条单图** 或 **A4 分页打印**
- **示例**：一键加载「生日快乐」

乐谱区 **滚轮** 上下抽拉；**Delete** 删除指向/锁定行；鼠标悬停可 **实时预览音高**（无需点击）。

## 振镜 / 扫描参数

| 参数 | 默认 | 说明 |
|------|------|------|
| 帧率（振镜） | 22 Hz | 设为 **0** → 按住鼠标滑动手动扫描 |
| 扫描线宽 | 200 px | 设为 **0** → 单点激光（沿移动轨迹采样） |
| 激光线宽 | 2 | 沿线移动平均窗口 |
| 平滑 Lerp | 0.5 | 扫描方向跟随平滑 |
| 振镜正反 | 开 | Barcoder 式来回拼接波形 |
| Barcoder 窗函数 | 开 | 段首段尾淡入淡出 |

**音高近似：** `f ≈ 帧率 × 线宽 / 条纹周期（px）`

乐谱竖纹使用 **小数周期** 精确对齐目标音高；条码演奏模式的竖纹/方框等使用整数像素条纹以保证 crisp 渲染。

## 操作速查

- **扫描发声**：按住鼠标在条码上移动（线宽 > 0 时显示红色扫描线）
- **线宽**：滚轮（乐谱模式下滚轮改为上下滚动乐谱）
- **振镜方向**：跟随鼠标 / 锁定横向 / 锁定纵向
- **乐谱锁定行**：**X**（再按取消）
- **乐谱删行**：**Delete**
- **图片长条**：滚轮或方向键上下滚动
